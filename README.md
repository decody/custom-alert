# custom-alert
Custom Alert and Confirm Modal with Vanilla Javascript

This custom alert version is currently test version. Probably v0.25

### usage: alert
```js
var modal = new Modal("alert", "이것은 경고창입니다");
```

### usage: confirm
```js
var modal = new Modal("confirm", "이것은 확인창일까요?");
```

### Plan
